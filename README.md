# PhytoLit AI 🦅

> Advanced Plant Pathology Literature Synthesizer and Research Assistant.

PhytoLit AI is a modern web application built for researchers, agronomists, and graduate students to synthesize peer-reviewed literature, format comparative data grids, export BibTeX citations, and interact with complex agricultural studies in real-time.

## 🚀 Features
- **AI-Powered Synthesis:** Generates comprehensive academic literature reviews, executive summaries, and key takeaways using Gemini.
- **Modern Split-Panel UI:** Sleek, glassmorphism dark-mode interface optimized for widescreen research workflows.
- **Interactive Chat Engine:** Drill down into specific findings with a built-in context-aware Q&A companion.
- **Export Capabilities:** Instantly convert outputs into presentation-ready PDFs or export `.bib` files for reference managers.
- **Local History:** Automatically saves recent search queries via browser local storage.

## 💻 Tech Stack
- **Framework:** Next.js (React / TypeScript)
- **Styling:** Tailwind CSS
- **AI Integration:** Google Gen AI SDK (`gemini-3.6-flash`)
- **PDF Engine:** html2pdf.js
